# Resume Portfolio

Vercel에 배포되는 개인 이력서 포트폴리오입니다.

## 빠른 시작

```bash
# 의존성 설치
npm install

# 개발 서버 시작
npm run dev

# 프로덕션 빌드
npm run build

# 배포
vercel deploy --prod
```

## 프로젝트 구조

- `app/` - Next.js 앱 디렉토리 (페이지, 레이아웃)
- `components/` - 재사용 가능한 React 컴포넌트
- `public/` - 정적 자산 (이미지 등)

## 배포

이 프로젝트는 Vercel에서 자동으로 배포됩니다. GitHub에 push하면 자동으로 배포되도록 설정할 수 있습니다.
